## Assignment operators

1. =	x = y	(вместо x = y)
2. +=	x += y	(x = x + y)
3. -=	x -= y	(x = x - y)
4. *=	x *= y	(x = x * y)
5. /=	x /= y	(x = x / y)
6. %=	x %= y	(x = x % y)
7. **=	x **= y	(x = x ** y)

8. <<=	x <<= y	(x = x << y)
9. >>=	x >>= y	(x = x >> y)
10. >>>=	x >>>= y	(x = x >>> y)

11. &&=	x &&= y	(x = x && (x = y))
12. ||=	x ||= y	(x = x || (x = y))
13. ??=	x ??= y	(x = x ?? (x = y))
